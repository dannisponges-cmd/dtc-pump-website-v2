# DTC Pump Website

A GitHub-ready Next.js website package for DTC Pump.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Create a new GitHub repository.
2. Upload/push all files in this folder to the repository root.
3. Import the repository in Vercel.
4. Framework preset: Next.js.
5. Build command: `npm run build`.
6. Output directory: leave empty/default.

## Important

Upload the contents of this folder to GitHub, not the zip file itself.
